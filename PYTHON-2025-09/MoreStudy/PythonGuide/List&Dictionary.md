# **List** (리스트)
리스트는 데이터들을 잘 관리하기 위해서 묶어서 관리할 수 있는 자료형 중의 하나


시험을 봤는데, A, B, C반 학생들의 시험 점수는 다음과 같았다.
- A반: 100, 20, 50, 70, 60, 90, 100
- B반: 55, 75, 50, 65
- C반: 90, 95, 100, 100, 100, 85, 15, 10, 5, 0, 3

시험 점수에 대해서 **각** 반의 **평균**을 구할 수도 있고, **전체** 반의 **평균**을 구할 수도 있고, **표준편차**를 구할 수도 있다.
데이터를 잘 관리하면 자료를 더욱 효율적으로 분석하할 수 있다.

A, B, C에 각 점수를 **하나로 묶어서** 저장해보자.

```
A = [100, 20, 50, 70, 60, 90, 100]
B = [55, 75, 50, 65]
C = [90, 95, 100, 100, 100, 85, 15, 10, 5, 0, 3]
```

**대괄호** []로 묶어서 **리스트**를 구현할 수 있다.

## 리스트의 연산
### 덧셈
리스트와 리스트간의 연산은 문자열과 같이 덧셈만 가능하다.
```
a = [1,2]
b = [3,4]
print(a+b)
```
`[1,2,3,4]` 리스트+리스트는 문자열+문자열과 마찬가지로 서로 이어붙는다.

### 곱셈
리스트에 정수를 곱할 수 있고, 그 수만큼 리스트의 내용이 반복된다.
```
a = [1,2]
print(a*3)
```
`[1,2,1,2,1,2]`

## Indexing
리스트의 처음 원소는 첫 번째가 아닌 **0 번째** 이다.
```
a = [10,20,30,40,50]
```
리스트 a에 대해서 
- `a[0]`은 `10`이다.
- `a[1]`은 `20`이다.
- `a[2]`는 `30`이다.
- `a[3]`은 `40`이다.
- `a[4]`는 `50`이다.
- `a[-1]`은 `50`이다.

## len()
리스트의 크기를 반환하는 함수.
```
a = [10,20,30,40,50]
```
리스트의 크기는 원소의 갯수와 같다. 리스트 a에는 원소가 5개 있으므로 크기가 5이다.
```
a = [10,20,30,40,50]
print(len(a))
```
위의 코드를 실행해 보면 `5`라고 출력되는 것을 확인할 수 있다.

# **Dictionary** (딕셔너리)
Dictionary는 리스트와 마찬가지로 데이터들을 잘 관리하기 위해서 묶어서 관리할 수 있는 자료형 중의 하나

키(key)-값(value) 쌍으로 이루어진 자료구조이다.

**키(key)**
- 키는 딕셔너리에 저장된 해당 값을 참조하는 데 사용되는 고유 식별자
- 키는 일반적으로 문자열이나 숫자이지만 튜플(불변 요소만 포함)과 같은 다른 불변 데이터 유형일 수도 있다.
- 딕셔너리의 각 키는 고유해야 한다. 기존 키에 새 값을 할당하면 이전 값이 대체됨.

**값(value)**
- 값은 딕셔너리의 해당 키와 관련된 데이터
- 값은 문자열, 숫자, 목록, 튜플 또는 기타 딕셔너리과 같은 모든 데이터 유형이 될 수 있다.
- 값은 딕셔너리에서 중복 가능. 여러 키가 동일한 값을 가질 수 있다.

```
person = {"name": "Alice",
          "age": 30,
          "city": "New York",
          "job": "Software Engineer"}
```
위의 예시에서 키는 "name", "age", "city" 및 "job"이고 값은 "Alice", 30, "New York" 및 "Software Engineer" 이다.

key와 value를 아래의 코드를 통해서 확인할 수도 있다.

### .keys()
```
# key 리스트 출력하기
print(list(person.keys()))
```
`['name', 'age', 'city', 'job']`

### .values()
```
# value 리스트 출력하기
print(list(person.values()))
```
`['Alice', 30, 'New\xa0York', 'Software\xa0Engineer']`

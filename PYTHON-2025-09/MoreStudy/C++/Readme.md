# 1. cout, cin, endl, string
- [Links](https://y-min.tistory.com/8)

```
#include <iostream>

int main() {
    std::cout << "C++ Code!!" << std::endl;
    return 0;
}
```
`C++ Code!!`

```
#include <iostream>

using namespace std;

int main() {
    cout << "C++ Code!!" << endl;
    return 0;
}
```
`C++ Code!!`

- `cin`: 데이터를 입력 받는다. (`>>` 기호를 사용함)
- `cout`: 화면에 데이터를 출력한다. (`<<` 기호를 사용함)
- `endl`: 줄 바꿈을 의미 (C언어 및 Python의 '\n'과 동일)

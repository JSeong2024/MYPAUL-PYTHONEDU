## 2025.09.23. 화요일
- 금일 수업을 진행한 결과 2주차부터 수업방식을 다음과 같이 변경하고자 합니다.
  - **코드 설계**와 **코드 해석** 두 파트로 나눌 것입니다. (개념강의, 과제, 테스트 포함. 자세한 내용은 2주차 수업에서 안내)
    - 코드 설계는 파이썬을 배우고자 하는 학생 특히 개발자 분야로 가려는 학생에게 도움이 될 것입니다.
    - 코드 해석은 자신이 코드를 직접 짜는 일은 적고 해석 정도만 할 수 있으면 된다고 생각하는 학생에게 도움이 될 것입니다.
  - 당일 수업에 필요한 **기초내용을 학습할 수 있는 자료**를 매 수업시간에 제공할 것입니다. 필요한 학생은 받아갈 수 있도록 할 예정. (가능한 온라인 게시판에도 업로드할 것입니다.)
  - 하나의 코드를 짜는 데 기초개념만을 사용한 방법부터 그 이상의 기법을 사용하는 방법까지 난이도별로 설명할 것이므로 본인의 판단하에 선택해서 이해하면 됩니다.

- 프로젝트 관련 2주차 수업까지 **팀(1인팀 가능)을 구성**하기 바랍니다.
- **과제**: **1주차 시험에 사용된 개념을 공부**해오시기 바랍니다. (7번문제 제외. 자세한 내용은 [과제 게시판 참고](https://github.com/JSeong2024/2025-MYPAUL-PYTHONEDU/tree/main/PYTHON-2025-09/Intermediate/Assignment))

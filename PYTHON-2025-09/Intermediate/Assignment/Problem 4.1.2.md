# **Problem 4.1.2. 피라미드 만들기**
## 배경 지식(Background)
```
for i in range(1,6):
  print('◆'*i)
```
위의 코드를 실행하면 결과는 다음과 같다.
```
◆
◆◆
◆◆◆
◆◆◆◆
◆◆◆◆◆
```
- `range(1,6)`: 1, 2, 3, 4, 5를 의미한다.
- `print('◆'*i)`: '◆'를 i번 복사해서 출력한다.
- 정수 n을 입력받는 코드는 `n = int(input())` 이다.
  
---
## Question

- 각 층마다 흰색(◇)과 검은색(◆)을 번갈아 사용해서 피라미드를 만들고자 한다.
- 꼭대기는 흰색(◇)을 사용한다.
- 2를 입력하면 다음과 같은 2층 피라미드가 만들어져야 한다.
```
◇
◆◆
```
- 5를 입력하면 다음과 같은 5층 피라미드가 만들어져야 한다.
```
◇
◆◆
◇◇◇
◆◆◆◆
◇◇◇◇◇
```
- 정수 n을 입력하면, 위의 조건을 만족하는 n층 피라미드가 만들어지도록 프로그램을 설계하시오.

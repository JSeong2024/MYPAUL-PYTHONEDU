# **Problem 5.1. 가위바위보 게임 만들기**

## 배경 지식
- 랜덤 함수를 사용해 '가위/바위/보' 중에서 무작위로 하나를 선택하도록 할 수 있다.
  ```
  import random
  
  comHand = random.choice(["가위", "바위", "보"])
  print(comHand)
  ```
  
## 문제
- 사용자는 '가위', '바위', '보'중 하나를 입력할 수 있다.
- 위의 코드를 활용해서 컴퓨터와의 가위바위보 게임에서 이기면 **승리**, 지면 **패배**, 비기면 **무승부**라고 출력되는 프로그램을 만들어 보자.

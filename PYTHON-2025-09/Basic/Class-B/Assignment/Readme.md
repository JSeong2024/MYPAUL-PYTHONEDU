## 2025.09.27. 토요일
**2주차 과제** 안내
1. 교재 챕터2 연습문제 p.74 ~ 75 (**1 ~ 10번 문제**) (11번 문제는 풀지 않아도 됩니다. 선택사항임)
2. 교재 챕터2 내용 복습
3. 교재 챕터3, 4 내용 공부해 올 것 (챕터3, 4 연습문제를 풀어 올 필요는 없음)
4. 2주차 주간 테스트 내용(1~5번 문제) 복습해 올 것

- **과제 1번**에 대해 2025년 **10월 17일 금요일 오후 11시 59분**까지 eins2017@naver.com으로 제출 바랍니다.
- 제출 방법
  - 정해진 양식은 없지만, 어떻게 만들어야 될지 감이 잡히지 않을경우 **예시 양식**을 참고하시기 바랍니다. ([예시 양식 링크 클릭](https://github.com/JSeong2024/2025-MYPAUL-PYTHONEDU/blob/main/PYTHON-2025-09/Basic/Class-B/Assignment/%EA%B3%BC%EC%A0%9C%20%EC%96%91%EC%8B%9D.hwp)) (hwp 한글 파일로 올려뒀는데 한글파일이 열리지 않는다면.. 힘내서 직접 만드시길 바랍니다)
  - 이메일 제출시 제목을 반드시 **클래스, 이름, 과제번호** 형식으로 해주시기 바랍니다.
    - 예를들어, 2주차 과제의 경우 ``초급B, 진우, 2주차과제``를 제목으로 하면 됩니다.
    - 주어진 과제들을 **한번에 제출**하기 바랍니다. (2주차 과제 중 제출해야 되는 과제는 1번으로, 1개만 제출하면 됩니다.)

  - 다음은 예시입니다. 챕터2의 경우 1 ~ 9번 문제는 객관식이고, 10번 문제는 코딩 문제입니다. 1 ~ 9번은 정답 번호만 적으면 되고, 10번에는 코드를 적거나 실행 결과가 같이 보이도록 캡쳐해서 업로드해도 좋습니다.

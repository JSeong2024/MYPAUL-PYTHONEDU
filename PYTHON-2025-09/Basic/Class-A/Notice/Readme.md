## 2025.09.21. 일요일
- **수업 속도**가 조금 **천천히** 진행되도록 조정할 예정입니다.
  - 코드 짤 시간을 충분히 드릴것입니다.
  - 대신 여러 난이도의 코드 문제를 제시할 것이고, 이전 단계를 빨리 마무리한 학생은 다음단계 코드를 진행할 수 있도록 할 것입니다.
- 2주차 수업 전까지 **설문조사**를 진행해주시기 바랍니다.
  - 링크: https://docs.google.com/forms/d/e/1FAIpQLSdoYDuR8HQn5eQDHr1Xm_JAeD7qg99p10X0QY0Aay3nqGy9Vg/viewform?usp=header
  - 파이썬 초급과정을 신청한 이유, 앞으로의 진로 계획 등을 알고자합니다.
  - 학생이 원할경우 진로와 관련된 자료(프로그래밍과 관련된 내용만)를 제공할 것입니다.
- 수업 후 설문조사를 통해 수업 환경을 꾸준히 개선할 것이므로 요청사항이 있다면 언제든지 알려주시기 바랍니다.

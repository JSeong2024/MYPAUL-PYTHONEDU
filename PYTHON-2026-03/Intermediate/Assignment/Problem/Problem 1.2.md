# **Problem 1.2. 리스트**
## 배경 지식(Background)

- **하나의 변수**에는 **하나의 값**만을 대입할 수 있다.
- `a = 10` 이 코드는 정상이지만
- `a = 10, 20` 이 코드는 오류가 발생한다.
- 하나의 변수에 여러 개의 값을 대입하고 싶다면, **여러 개의 값을 하나의 값으로** 만들어주면 된다. → 즉, **하나로 묶어**주면 된다.
- 값을 **대괄호**로 묶어서 하나의 값으로 표현하는 자료구조가 **리스트**이다.
```
a = [10, 20, 30, '40', '50']

print(a[0], a[1], a[4])
```
위의 코드를 실행하면 결과는 다음과 같다
```
10, 20, 50
```

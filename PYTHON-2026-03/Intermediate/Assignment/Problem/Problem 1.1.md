# **Problem 1.1. 시간 분 초**
## 배경 지식(Background)

- 5000초는 몇 시간 몇 분 몇 초일까?
- 다음과 같은 코드를 설계할 수 있다.

```
second = 5000

h = second//3600
m = (second%3600)//60
s = (second%3600)%60

print(f'{h}시간 {m}분 {s}초')
```
위의 코드를 실행하면 결과는 다음과 같다
```
1시간 23분 20초
```

원리를 살펴보자.

5000초에는 '**시간**', '**분**', '**초**'가 들어있다.
  
먼저 '시간'을 뽑아보자.

5000초에 1시간이 몇 번 있는지는 5000초를 1시간으로 나눈 **몫**을 구하는 것과 같다. → `h = second//3600`

5000초를 1시간으로 나누면 몫과 나머지가 발생하고

몫이 '시간'을 의미하므로 **나머지**에는 '시간'을 제외한 '**분**'과 '**초**'가 존재함을 알 수 있다. → `second%3600`

이것을 다시 1분으로 나눈 **몫**이 '**분**'이 되고 → `m = (second%3600)//60`

그 **나머지**가 '**초**'가 될 것이다. → `s = (second%3600)%60`

## Question

- **하루**는 **24시간**이다.
- **1시간**은 **60분**이다.
- **1분**은 **60초**다.
```
second = 300000
```

- **300,000초**(30만초)는 **며칠** 몇 **시간** 몇 **분** 몇 **초**인지 구해보자.
- 30만초를 **일**, **시간**, **분**, **초** 단위로 나누는 프로그램을 설계하시오.

# **Problem 3. 합격? 불합격?**
## 배경 지식(Background)
### 리스트와 반복문
- 리스트와 반복문을 활용하여 **많은 양의 데이터를 한 번에 효율적으로 처리**할 수 있다.
  
  ```
  a = [10, 20, 30]
  for i in a:
    print(i)
  ```
  `
  10
  `
  <br />
  `
  20
  `
  <br />
  `
  30
  `
  <br />
  <br />
  
  ```
  a = [10, 20, 30, 40]
  total = 0
  for i in a:
    total += i
  print(total)
  ```
  `
  100
  `

## Question

- 점수에 따라 합격, 불합격 여부를 판단하는 프로그램 설계.
- 점수(score)는 리스트 형태로 주어진다.
- 70점 이상이면 **합격**
- 70점 미만이면 **불합격**

 **예시**
  - `score = [80, 65, 70, 95]`
  - 80점은 70점 이상이므로 **합격**
  - 65점은 70점 미만이므로 **불합격**
  - 70점은 70점 이상이므로 **합격**
  - 95점은 70점 이상이므로 **합격**

  - 따라서 최종적으로 다음과 같이 출력된다.
    ```
    합격
    불합격
    합격
    합격
    ```

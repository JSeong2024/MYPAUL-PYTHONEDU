- 제출 방법
  - 정해진 양식은 없지만, 어떻게 만들어야 될지 감이 잡히지 않을경우 **예시 양식**을 참고하시기 바랍니다. ([예시 양식 링크 클릭](https://github.com/JSeong2024/MYPAUL-PYTHONEDU/blob/main/PYTHON-2026-03/Basic/Assignment/%EA%B3%BC%EC%A0%9C%20%EC%96%91%EC%8B%9D.hwp)) (hwp 한글 파일로 올려뒀는데 한글파일이 열리지 않는다면.. 힘내서 직접 만드시길 바랍니다)

  - 다음은 예시입니다. 챕터2의 경우 1 ~ 9번 문제는 객관식이고, 10번 문제는 코딩 문제입니다. 1 ~ 9번은 정답 번호만 적으면 되고, 10번에는 코드를 적거나 실행 결과가 같이 보이도록 캡쳐해서 업로드해도 좋습니다.
